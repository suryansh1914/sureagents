# SureAgents Review (Kiro)

Run:

```bash
SUREAGENTS_ORIGIN=kiro-cli sureagents review
```

You may append an optional PR URL:

```bash
SUREAGENTS_ORIGIN=kiro-cli sureagents review <pr-url>
```
